## 💻 How to Run
To run any of these files on your local machine:
1. Ensure you have a C++ compiler (like G++) installed.
2. Clone this repository: `git clone [your-repo-link]`
3. Navigate to the folder: `cd [folder-name]`
4. Compile the file: `g++ filename.cpp -o output`
5. Run the program: `./output`
